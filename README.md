# Meu Portfólio

Esse é um projeto **Angular CLI** na versão 20.0.0.
Junto ao Angular, utilizei o framework **Boostrap** para facilitar a estilização.

## Sobre o projeto

Esse é um projeto feito para apresentar meu perfil como **Desenvolvedora Web** focado no front-end e suas vertentes, desde o design pensado em UX/UI e o desenvolvimento do projeto.

A fim de apresentar os projetos feitos ao longo dos meus estudos e praticar o uso do framework Angular, o portfólio foi desenvolvido para apresentar meu perfil profissional.

Antes do desenvolvimento com as tecnologias, foi pensado um **protótipo** exclusivo e personalizado no meu gosto pessoal para montar o design do final do projeto.

O portfólio foi criado utilizando a ferramenta design **Figma**, onde você pode visualizar pelo link: [Protótipo no Figma](https://www.figma.com/design/pCBPfsCNb6h7vEg0i9Pdsh/Meu-Portfolio-2025?node-id=0-1&t=weAmhQdy7UpvyBFf-1)

Ao criar meus projetos e participar da criação deles durante a faculdade, gosto de seguir esse padrão de montar o design e após concluído, partir para o desenvolvimento. Pois assim facilita o momento de criação e consigo ter um amontoado de ideias práticas e visualmente agradáveis, deixando o projeto mais organizado.

## Pensamentos futuros

Esse projeto ainda está em desenvolvimento, então continuo trabalhando em melhorias e refatoramento do código. Posteriormente irei aplicar a responsividade e adicionar mais projetos conforme minha produção.
